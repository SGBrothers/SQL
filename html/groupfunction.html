AGGREGATE FUNCTIONS/GROUP FUNCTIONS

Aggregate functions are also called Group functions are built-in sql functions that
operate on groups of rows and return one value for entire group table or table.

Types of Aggregate functions :-

(i) count()

(ii) sum()

(iii) max()

(iv) min()

(v) avg()

Count

This function returns the number of rows in the table that
satisfies the condition specified in the where clause.

Example :- select count(*) as total_student from student where marks1=80;

Sum

This function is used to get the sum of a numeric column

Example :- select sum(salary) as salary_budget from employe;

Max

This function is used to get maximum value from column

Example :- select max(marks1) as maximum_marks from student;

Min

This function is used to get minimum value from column

Example :- select min(marks1) as minimum_marks from student;

Average

This function is used to get average values of a numeric column

Example :-select avg(marks1) as average_marks from student;
